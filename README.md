## Usage
環境はDockerにより構築する．任意の場所でbuid & run可能なはず．  
To build & run:
```
cd Docker
sh build.sh
sh run.sh
```

#### Note :
  - The environment runs on GPU so you must install NVIDIA Container Toolkit.
https://docs.nvidia.com/datacenter/cloud-native/container-toolkit/latest/install-guide.html
